- In CLI:

* npx tailwindcss init
* npm init -y
* npm i -D prettier-plugin-tailwindcss
* npm run tailwind

- In "Scripts" of package.json:

* "tailwind": "npx tailwindcss -i ./src/input.css -o ./build/css/style.css --watch"
* "prettier": "npx prettier --write \*_/_.html"
